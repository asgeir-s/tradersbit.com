API

This API can be used to publish trading signals to an already created stream.

Endpoint
              

https://api.tradersbit.com/api/streams/'stream id'/signal
Method
              
POST
Headers
  • Content-Type: application/json
  • x-authorization: apikey 'streams apikey'
Api keys can be generated in the My Streams view (under Publisher). Click on the three-dot icon on one of you streams (icon on the far right of you streams on the My Streams page).
Body
The body of the request should be -1 (SHORT), 0 (CLOSE) or 1 (LONG).

Example
                
POST:
https://api.tradersbit.com/api/streams/4b0c2989-ca71-46c5-a138-c4777a41e13e/signal
Headers: 
                
Content-Type: application/json
x-authorization: apikey eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJteUJhbmtQaW4iOjg...
Body: 
1
