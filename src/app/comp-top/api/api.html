API

This API can be used to publish trading signals and get a streams current staus (last signal).

Api keys can be generated in the "my streams" view (under +Publish). Click on the gear icon on one of you streams.

Post signal

Endpoint
      

https://api.tradersbit.com/api/streams/'stream id'/signal
Method
      

POST
Headers
      

Content-Type: application/json
x-authorization: apikey 'streams apikey'
Body
The body of the request should be -1 (SHORT), 0 (CLOSE) or 1 (LONG).

Example
Request 
                

POST:
https://api.tradersbit.com/api/streams/4b0c2989-ca71-46c5-a138-c4777a41e13e/signal
Headers 
                
                  
Content-Type: application/json
x-authorization: apikey eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJteUJhbmtQaW4iOjg...
Body 

1

Response 

              
[
  {
    "timestamp": 1452764561408,
    "price": 429.49,
    "change": 0.0052428321,
    "id": 25,
    "valueInclFee": 0.9749317064,
    "changeInclFee": 0.0032428321,
    "value": 0.9735364263,
    "signal": 0
  }
]

Get status

Endpoint
      

https://api.tradersbit.com/api/streams/'stream id'/status
Method
      

GET
Headers
      

Content-Type: application/json
x-authorization: apikey 'streams apikey'
