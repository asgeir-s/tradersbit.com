  • The competition leaderboard now reflect the competition rule of minimum 10 closed trades.

  • Publishers can now change the subscription price for theire streams.
BETA 0.1.8 (11 Feb 2016)
  • Publishing is fixed for Firefox.
  • Smoother browsing experience on mobile.

  • Subscribers get a confirmation email. The mail includes the stream's current position.
  • Publishers get a notification email when they sell subscriptions.
  • Some small UI changes for mobile and some fixed typos.

  • Added fees for both opening and closing positions in "unrealized PL" in the "publisher-dash" and notifications. This is only a interface change. Thus, it does not affect any statistics.
  • Fixed profit charts.
  • Enhanced signup, sign in and password rest flow.
  • Change date format to a universally understandable format.
  • Top stream at landing page is now the streams with highest Net Profit.
  • Fixed calculation of indicator 'Net Profit Excluding fees'.
